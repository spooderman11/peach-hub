Game Detection Interface

When you launch our main script, you will be greeted with a UI. that ui is our innovative game detection interface. with the detection interface, you can instantly pull up a script for your favourite game. if the game you chose doesnt have a supported script, then you can choose to execute a universal script. the universal script is powerful in every way, just like scripts that are originally made for a game.

Some game detection interface ui works like any other interface ui, execute, press a key, then go. our ui works differently. we check to see if your game is supported in the background, if it is not, the ui will show up and then disappear. if it is, then you can press a keybind and have your script running in seconds!
